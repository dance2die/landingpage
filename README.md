## How to run

Make sure that you have the Gatsby CLI program installed:

```sh
npm install --global gatsby-cli
```

Clone the repo.

```sh
git clone https://github.com/dance2die/landingpage.git
```

Then you can run it by:

```sh
cd landingpage
gatsby develop
```
